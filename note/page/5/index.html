2011年的零碎片段

2011-11-02 # 随手记

2011-11-10

湖南艺术职业学院其实是由湖南省艺术学校和湖南省电影学校合并而成，但湖南省艺术学校和湖南省电影学校也各自有很漫长的成长历程。湖南省艺术学校最初成立于1951年，当时也是叫湖南省艺术学校，后来在1953年的时候改为湘剧小演员训练班，在1957年时改为湖南省戏曲学校，1959年为湖南艺术学院，1961年为湖南省戏曲艺术学校，1964年为湖南省戏剧学校，1976年复名为湖南省艺术学校。

湖南省电影学校最初成立于1979年，当时叫湖南省电影技工学校，1985年改为湖南省电影职工中等专业学校，1994年正式命名为湖南省电影学校。我入校时，母校叫湖南省电影学校。毕业那年，湖南省电影学校和湖南省艺术学校合并，升格为现在的湖南艺术职业学院。

转眼间已经毕业12年了，时间过得真快啊。湖南艺术职业学院建校60周年庆典活动将于11月18日举行，上午8:30在左家垅校区（原湖南省艺术学校）和望城坡校区（原湖南省电影学校）由各系分别举行校友联谊活动。下午14:30在贺龙体育馆（长沙天心区劳动西路339号）隆重举行校庆庆典大会暨《母校·摇篮》（暂定名）大型文艺演出活动。届时，张也、甘萍、吴军、瞿颖、雷佳、王丽达、周浩等一大批著名校友将登台演出。我们热忱期盼海内外校友、各级领导、各界人士热情支持和参与校庆活动，共襄盛典，同谱华章！

2009年的零碎片段

2009-10-01 # 随手记

2009-02-02

我已经有很久没有回家看望自己的父母，非常想念她们。所以公司刚宣布放假，我就立马飞奔汽车站，买好了返乡的车票。由于锦涛哥这几年带领团队精诚合作，城市及乡村的道路建设也随之飞速接轨，可以看到主干马路的各村口交叉部分都是水泥硬化进村，以前可都是些黄泥或者水洼。正因如此，返乡的旅程不再如以前那般辛苦，只需转乘三两趟车就到家了。要是以前，那可是至少要转五趟车，而且还买不到车票，只能在拥挤的车上站六小时。

正如公司一位领导曾经说过的，大部分人一回家就会说：“妈妈，我回来了。”看来我也是那群人之一，因为我也是这么做的。

家里的黄狗居然没有冲我狂吠，而且还晃动着尾巴对我撒娇，看来狗真的通人性，时隔两年多仍然记得我是自家人。不过它还是没有放弃老习惯，仍然是使用左前腿与我握腿，要是哪天它首先伸出右腿的话，我一定要奖励它更多。

和妈妈一起迎接我的还有刚满四个月的小侄女，这个非常认生的小家伙居然愿意让我抱着并且不哭不闹。这一点让妈妈还有哥哥嫂嫂都闹不明白，我想要么是我看起来非常面善，小朋友不怕我，要么就是小家伙知道我给她带了小礼物。哈哈...谁知道呢。

爸爸正在村口的一户人家做活，说要争取在过年前帮他们将地砖弄好，要让别人过个舒服年。他只是抬起头来笑着打了个招呼，并没有特别的表示。老爸就是这样，他在孩子的面前永远是一副理性坚强的样子，但他却和妈妈一样，是一个非常疼孩子的人。记得两年前我回长沙的时候，妈妈为了避免自己不舍得我的离开会哭泣，叫爸爸去送我，结果我刚上车，爸爸就赶紧扭过头去擦拭眼泪。

哥哥的变化挺大，成熟了很多。做事情的时候不再鲁莽行事，毕竟他已为人父，懂得仔细地思考问题。但身材却一点没变，仍然是两年前的那个大肚腩，体重比我超出二十公斤。要是哥俩再交手，估计我只能选择三十六计之走为上，因为我和他不是一个重量级的。当然，不会有这样的机会。咱俩虽然常年不通电话，但见面后并不觉得有什么隔阂。兄弟嘛，就是这样，不管多久不见面不通话，依然不改血浓于水的兄弟情义。

这是第一次见嫂子，所以没有传统记忆做对比。不过听妈妈说，嫂子为了能够让我们这一家子有个大团圆，早在一个月前就抱着宝宝回到娘家，前段时间刚回来。不过她的这番美意并没有实现，因为我是一个人回家的。不过，这已经是很难得的，我们一家子可是足足有三年多没聚到一起了。

刚回家两天的样子，姑姑打电话说奶奶快不行了，希望我们过去看看，万一她真的去世了，也不会留下什么遗憾。奶奶和爷爷以及他的那家子与我家在二十多年前就有一些误会，这场误会导致爷爷奶奶从来就没有照顾过我和哥哥，他们还阻止同村的长辈照顾我们。在农村里，没有人照顾小孩，大人就无法安心耕作，像我老爸这种在外地工作的人更不能安心做事。正因为这样，爸妈决定将我托付给亲戚照顾，所以我现在偶尔还会想起桂林、云南、永州等地的一些记忆痕迹。不过，我那时候不超过两岁，又能记得一些什么呢？记忆最深刻的要数刚回到爸妈身边的那个时候，我居然不认识爸爸妈妈，管他们叫哥哥姐姐。我不知道爸妈当时的那种心情，我想其中应该会包含一丝丝的酸楚吧。

尽管如此，我们还是决定乘坐六小时的汽车去看看她老人家。毕竟人之将死，我们也就不要再去理会那些恼人的误会了吧。这叫“搁置争议”

情况似乎很糟糕，姑姑眼睛通红，满手都是碘酒的颜色。因为她是一位医生，她将奶奶接到她所在的医院宿舍，照顾起来更方便，什么氧气等大型护理器械都能够搞定。进屋后，看到了躺在床上的奶奶，她已经不能说话，不过尚有意识。当听到我们声音的时候，她一只手伸向天空，头往后仰，我们所有人都已为她就要殁了，还好没事。

2009-05-22

一个员工轻轻地走进办公室，小声地对我说：“海宇哥，我想辞职！”

我并没有太在意，在经过简短的询问之后，我大致同意了她的请求。我对辞职这种事情向来抱有一种态度：只要员工有合适的地方去，比我这里更好，能够学到更多东西，得到更多工资和福利等待遇，我一般会同意离职。

接下来的几个小时后发生的事情让我很难受，也让我自我反省没有关注一线员工在思考什么问题。

那是快要下班的时间，大概在三点半到四点钟左右，另一个员工也是极其轻巧地步入办公室，以更小的声音对我说：“海宇哥，我要辞职！”… …

今天早上，接到一条短信：“海宇哥，我有件事情和你说，我们发短信说咯，不敢和你打电话说”。我一看，心里咯噔一下，我知道这可能会意味着什么，于是发过去四个字：“你要辞职？”果不其然，正是！

除这三个员工之外，还有一位员工需要调职，这是比较靠前的一件事情，但毕竟还是要走的。这三四个员工辞职、调职等事情堆积在一起，肯定是有什么问题存在。如果不是，也不会在刚刚提升工资奖金方案一两个月之后集中爆发。

今天好好休息，调整思绪。明天逐个聊聊，看问题出在哪里。
