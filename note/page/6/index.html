2008年的零碎片段

2008-10-03 # 随手记

2008-01-28

很多人和我一样喜欢冬天里洁白的雪，也希望有一个漫长雪季的冬天，这回上帝听见了这个愿望，并将它得以实现。一切都太突然了，没人会预料到这个雪季会持续大半个月，也没人会料到这场雪带来是持续冰冻。

一定要记住2008年的这次大雪

2008-06-07

以下是在和谐号D字头动车上忍受煎熬的时候用手机写的，看起来思维很散，不做整理了。

速度和准时是它最大的优势，这一点值得肯定。创造这个奇迹的原因之一是列车的上下车时间仅有1分钟，是否能在列车的车门处设立一个液晶显示屏幕？一旦列车到站就进行60秒的倒计时，方便上下车的乘客。另外我们几乎找不到乘务人员，因为我还没有看到他们。如果每个车厢都设有乘务员专座，随时侯着，就不会有人偷偷地在过道吸烟了。

应该在座椅后背加贴座椅使用说明书，方便第一次搭乘列车的乘客了解其功能，并最大化地享受乘车的快乐。

座椅太高，不能通过调节垂直仰度来改善，虽然可以进行水平的360度旋转，但这个功能仅适合那些坐车还开会的大忙人以及那些靠打牌来消磨时间的人。另外，想要水平转动三人座的座椅还得打扰前后排的客人，否则无法转动，看来这个功能还需改善啊。

它的洗手间也是洗漱室，但因为洗手间的狭窄构造使得盥洗盆只能设计太狭小，不方便洗漱，容易将水洒落在地板上，会使人误解为前任使用者小便时目标不明确或尿压太强，搞得一地邋遢。另外，我估计最终采用这种没有专用洗漱间设计的原因，可能在于D字车的定位。也许它被定义为快速商务列车，每次列车行车时间不会超过24小时，乘客不需要洗漱，只要有个洗手的地方就已经足够。这仅仅是猜想，还需要资料来证实。

乘客方面，除了有人偷偷吸烟之外，也有些人脱掉了鞋子，将大脚随处摆放，估计是想用自己臭脚来熏晕别人，以便其事后行事。还有人不用耳麦听音乐看电影，似乎要让所有人知道他带了笔记本电脑，似乎要让所有人知道她的手机是带有音乐播放功能的。我实在是受不了那份罪，体验上还不如来时的卧铺好。如果要将D头列车定位为与飞机媲美的运输品牌，还需努力，不过票价已经很接近了。

2007年的零碎片段

2007-02-25 # 随手记

2007-02-25

农历初六的晚上，卢哥打电话告诉我：一个只有21岁的同事因为心脏有问题...走了。

我怎么也不相信这件事情是真的，曾经在一起工作生活的那个小伙子平时不是挺好的吗？怎么突然就没了？

曾经将死看得淡如水的我如今胆怯了，死亡也许对逝者来说并不可怕也并不悲伤，但死亡却给逝者的亲人和朋友带来无尽的伤痛。它让生者明白，必须要好好的珍惜现有的生活和活着的每一瞬间，善待自己和亲人以及朋友！

2007-05-13

本来一直没打算安装电话和宽带，因为大部分时间都是在公司里度过的，回家之后除了玩游戏或特殊事情外，一般都不会用电脑；而电话方面都是用移动电话，所以也觉得没有安装电话的必要。不过，公司老大赞助我装宽带，所以还是决定去电信营业厅新装电话和宽带。预计六月份就可以在家里上网了。

另外，关于电话号码的凶吉问题倒是挺吓人的，亮哥[1]说去世的那个同事手机号码尾数为4。听到之后，我只觉得背后一凉。虽然不太相信这些，心里却有些担心和害怕。找到测试电话凶吉的网站一试，发现周围的圈子用的都是凶凶的号码，大家也都挺好！无所谓啦！

2006年的零碎片段

2006-01-09 # 随手记

2006-01-09

2005年过得比以前任何一年都要快许多，发生的事情也挺多的，庆幸的是很多事情都在慢慢的往好的方向发展。

只是我还没有去见拜见她的父母，而她也没去我家。不过，这并不能造成什么坏的影响。

今年在公司里做琐碎事情比较多，但没有给自己的部门带来什么盈利，也未曾看到明显的部门成绩，这是我最惭愧的。所以每次见到那几位老总都觉得自己很不好意思，无颜相见似的... ...

做得最多的琐碎事莫过于去外展点做蜂王[1]，公司的外展活动我基本上都得出去帮忙，所以跟大学生在一起的机会也就多了很多。也常常被这些还未进入社会单纯的他们所感染，发觉每个人都是那么热情洋溢，即便是有困难也能够相互帮助以完成预定的任务。精神难能可贵，有时候也会让我振奋不已。

另外，第一次听到一些公司的销售经理说我的作品很受顾客欢迎，这是我最高兴的事情了。所以每次他们点名要我帮忙做活动用的新样品时，总会带着那份幸福及压力去认真做好。

在婚纱摄影设计方面的改变主要是摒弃了以往以构图为重点的思想，将精力放在提升原照片的色彩及氛围上，同时将所要设计的框类产品的本身色彩考虑了进去，而构图则次之... ...

摄影方面还是没有进步，一直使用松下FX5这款傻瓜数码机，一般是拍些记录生活的片子，不过也有一些还不错的照片就用做素材了。这个照片还是要坚持拍的... ...

下半年明显的感觉到身体不如从前，所以每个星期都会去打打羽毛球，未曾进步，但手臂倒是粗了不少。体重也在望期望的目标上升... ...

忽然被很多东西给堵塞住了，想说但理不清头绪，罢了，不说了。认真踏实的做人做事吧...

2005年的零碎片段

2005-09-15 # 随手记

2005-08-17

很喜欢青年视觉，特别是那些陈逸飞先生在青年视觉做总编的时候的青年视觉。总感觉有东西可以看，有东西可以学。但陈逸飞先生走出青年视觉之后的那几期杂志让人很失望，与之前的杂志相差甚远，感觉上很粗糙，不细致。故有很长一段时间没有去i理会，没有将它带回家。

不过近来几期做得还不错。觉得比起陈先生离开杂志社之后那一段时间的杂志办得要好些，但还是不够精致。不过不可否认，在国内这本杂志还是很吸引年轻人的眼球的。

2005-08-20

上午的时候有点麻烦事情找过来，我不想处理，因为我认为别人也能够处理好。

不过，如果追续根源，那些都是我的事情。因为大约在二三个月前，我曾答应一位领导帮她为一个朋友设计相册。在经过三四次的反复修改之后，客人终于同意输出。然而现在，她却要求重新修改一些东西，当然那是很简单的事情(去掉相片上的一些中英文字)

so…我觉得拍摄照片的那个店的设计人员应该可以照葫芦画瓢能很容易搞定这件事情，让他们自己去修改吧，我这边正好腾不开手脚。但店里面却一直不接，说是他们做不出来。这时我却自以为是的说："这么简单的东西难道他们就做不出来？如果做不出来在这里干吗？应该要给他们机会做这些，要不然他们将永远得不到提升。"我知道在店里面的同事听到这些话肯定不怎么舒服，这应该也算是伤害了他们吧。但店里面的人说的比我更难听——“做不出是他们能力不够，后面只指派了这些人过来，那没有办法，如果他们没有得到提升就只好在将来被淘汰。”
