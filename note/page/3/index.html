2022年的零碎片段

2022-10-06 # 随手记

2022-01-03

利他的力量无穷闪耀，让很多的决定不再犹豫，从此内心有光！

2022-02-01

一个有意义的目标必须能让我们提升自己。这就意味着我们必须离开舒适区。

如果你明确知道一个目标应该怎么实现，那说明这个目标还是太简单了。

2022-03-25

一大早，想起要给同一天过生日的馨凤买鲜花，倒是她给我买的蛋糕先送到，应该是她比我先行动。我没留意看她从楼下拿过来的是什么，就问她鲜花还行吧，结果她一头问号地说那是她给我买的蛋糕。我这尴了个尬的……

岳母和母亲很早就在微信上发来问候。夫人也打来祝福电话，问我晚上怎么安排，小家伙们可是等着要吃生日蛋糕的。我还不好答复，因为我不知道接下来还有没有其他的安排。

中饭是志伟老师安排的，生日前一天她与美艳两人碰巧要我的生日算塔罗牌，结果惊呼生日岂不是明天？就算我说我过生日是过的农历也没糊弄过去，因为今年的生日是农历和公历在同一天。长沙的五一广场因为疫情原因还在管控中，很难得找到可以堂食的饭馆，她在河西找了一家店，还提前支付了餐费。然后我们几个人和同一天生日的同事开着车从河东跑到河西去吃饭了。

要说关注细节和安排招待，我得向她学习。

飞哥在家休息，特地安排了一大桌子的好菜，约上公司的八九个同事陪我过生日。豆豆为此还请了半天假给飞哥帮忙，先桂也是老早就跑到飞哥家里去打下手了。我笑称这不打两次麻将送点红包给他都过意不去。

大公主和小宝也让我感到惊喜。因晚上已确定在外吃饭，于是就点了蛋糕送回了家，让夫人和孩子们一起享用。大宝在吃蛋糕之前，特地把蜡烛点起来，打了一个视频电话让我许愿、一起吹蜡烛，仪式感满满。重点是她明白这生日蛋糕的意义，不只是为了吃蛋糕。

这一整天被人关爱着，沉浸在爱的世界里，感受别人对我的关爱，幸福感满满。让我不禁思考：“如何才能让公司的每一位伙伴都能感受被爱的感觉呢？”这将是我接下来需要落实的、非常重要的一项工作。

2020年的零碎片段

2020-10-01 # 随手记

2020-10-05

一个真正成熟的管理者看谁都顺眼，承认并且接受真正的多样性。

看顺眼并不代表认同，不代表认为对方就是完全正确的，并不代表“换成是我，也会这么做”。他可能一辈子都不会那么做，但是他知道这是站在对方的角度后，对方的选择，对方的价值观，对方的逻辑。

一个人同时保有两种相反的观念，还能正常行事，这是第一流智慧的标志。

这点小事在她看来非常重要，因为她学会了一个新的交流方式。昨天还特地发消息告诉我，她又想起来怎么发文字消息了。 ​​​

2020-10-20

要懂得止损

2019-06-13 # 随手记

借钱

前东家的老板一大早就找我借钱，说是急用做周转，半个月内就还给我。我下意识地想起，去年他也找我借钱给他救急，我刷了5张信用卡凑了几万块钱借给他，不计利息的那种，到目前为止还没有着落。

我知道他现在的境遇，每年各种节日前后都会有人到处找他，现在正好是端午节前，应该是有人要他还钱了。

盘了下自己有多少可以动用的资金，发现把银行卡里和微信钱包里的钱加起来都不够。最后只好如实相告，我帮不了他，也因没能帮到他而表示歉意。

我顺便想起曾经在去年的同一时期借了钱给另外一个同事，到现在还没有还，也没和我说什么，似乎根本就没有发生过借钱的事情。
